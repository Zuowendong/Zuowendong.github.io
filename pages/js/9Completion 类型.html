    • Completion 类型

    语句是任何编程语言的基础结构，与 JavaScript 对象一样，JavaScript 语句同样具有“看起来很像其它语言，但是其实一点都不一样”的特点。比较常见的语句包括变量声明、表达式、条件、循环等，这些都是作为研发非常熟悉的东西。

    这里介绍一下 JavaScript 语句执行机制涉及的一种基础类型：Completion 类型。

    # Completion 类型

    提问：在 try 中有 return 语句，finally 中的内容还会执行吗？

    function foo() {
    try {
        return 0;
    } catch (err) {
    } finally {
        console.log("a");
    }
}

console.log(foo());

    通过如上代码可以看到，finally 确实执行了，而且 return 语句也生效了，foo() 返回了结果 0。

    虽然 return 执行了，但是函数并没有立即返回，又执行了 finally 里面的内容，这样的行为就和预期有很大出入。

    还是上述例子，在 finally 中加入 return 语句，会怎样呢？

        return 0;
    } catch (err) {
    } finally {
        return 1;
    }

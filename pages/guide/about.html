      # vuepress 安装使用

      • 安装 node 环境，安装 yarn
      npm install -g yarn
      1
      • 新建项目文件夹，该路径下启动终端输入
      yarn init
yarn add -D vuepress
      1
      2
      • 进入项目，找到 package.json，添加一些 scripts
      "scripts": {
    "dev": "vuepress dev docs",
    "build": "vuepress build docs",
  },
      1
      2
      3
      4
      • 创建第一篇文章
      mkdir docs && echo '# Hello VuePress' > docs/README.md
      1
      • 本地启动服务，在终端输入：
      yarn dev
      1

      至此，在浏览器输入 http://localhost:8080，可以查看你的页面了：

      hello-vuepress

      但这只是个单独的页面，只是说明了项目已经搭建完成，能够成功运行了，远远不是个博客。
